# Logs Directory

This directory is used for locally generated execution logs produced during AI Lab testing.

These logs may include:
- prompt content
- model outputs
- timestamps
- local system and environment details
- experimental notes relevant to model behavior

 Important Usage Notice

Do not commit or upload logs that contain:
- personal data
- client information
- proprietary or confidential material
- credentials, tokens, or secrets
- sensitive operational details

Users are responsible for ensuring that any logs committed are safe for sharing.
When in doubt, do not commit the log.
